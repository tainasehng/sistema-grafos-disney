# Sistema de Grafos – Simulação Disney+

Projeto desenvolvido em Python utilizando estruturas de grafos para simular a organização e navegação de conteúdos.

## Funcionalidades
- Estruturas de grafos
- Algoritmos de navegação
- Organização de dados

## Tecnologias
- Python

## Objetivo
Aplicar conceitos de estruturas de dados e lógica de programação.
